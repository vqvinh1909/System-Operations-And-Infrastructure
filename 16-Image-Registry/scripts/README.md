# Scripts — Module 16: Image Registry

Thư mục này dùng để lưu các script bạn tự viết liên quan tới module này, ví dụ:

- Script tự động tag + push nhiều image cùng lúc lên registry nội bộ.
- Script gọi Harbor API để tạo project/robot account hàng loạt.
- Script kiểm tra dung lượng đĩa của registry và cảnh báo khi gần đầy.

Đặt tên file rõ ràng theo mục đích, ví dụ `push-all-images.sh`, `harbor-create-project.sh`, `check-registry-disk.sh`.
