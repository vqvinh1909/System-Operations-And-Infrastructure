# Labs — Module 18: Security

Thư mục này dùng để lưu toàn bộ code/config bạn tự viết khi làm theo [[../05-Labs|05-Labs.md]] của module này.

Gợi ý cấu trúc:

```
labs/
├── lab1-rootless-docker/
│   └── notes.md
├── lab2-capabilities/
│   ├── docker-run-examples.sh
│   └── notes.md
├── lab3-seccomp-apparmor/
│   ├── custom-seccomp.json
│   └── notes.md
└── lab4-scan-bench/
    ├── trivy-report-before.txt
    ├── trivy-report-after.txt
    ├── docker-bench-before.log
    ├── docker-bench-after.log
    └── notes.md
```

Hãy lưu lại `notes.md` cho mỗi lab ghi chú lại: lỗi gặp phải, cách bạn tự sửa, và điều bạn hiểu thêm sau khi làm — đây là tài liệu ôn tập giá trị nhất khi bạn quay lại xem sau vài tháng. Với Lab 4, việc lưu lại báo cáo scan/Bench trước và sau khi khắc phục là bằng chứng thực hành có giá trị khi phỏng vấn.
