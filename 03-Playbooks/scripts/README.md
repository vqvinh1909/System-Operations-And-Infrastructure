# Scripts — Thư mục script tự viết

Đây là nơi lưu **script hỗ trợ tự viết** (bash, python, ansible.cfg mẫu, alias...) khi học module "Playbooks".

Gợi ý tổ chức:
- Đặt tên script mô tả rõ mục đích (ví dụ `check-syntax.sh`, `generate-inventory.py`).
- Ghi chú ngắn ở đầu mỗi script: dùng để làm gì, chạy như thế nào.
- Đây là nơi tích lũy "bộ công cụ cá nhân" — càng học lên các module sau càng nên quay lại tái sử dụng script cũ.
